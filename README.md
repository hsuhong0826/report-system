# 資訊管家 - 企業級全方位 IT 報修管理系統

## 1. 產品願景
**資訊管家** 不僅僅是一個報修工具，它是企業數位化轉型的第一步。我們致力於打造一個**「零阻礙」**的 IT 服務環境，透過標準化、自動化的流程，將員工從繁瑣的設備故障回報中解放出來，同時賦予 IT 團隊更強大的管理與回應能力。

---

## 2. 我們解決的企業痛點

在傳統的企業 IT 支援中，您是否常遇到以下問題？

*   **資訊傳遞斷層**：員工透過電話或口頭報修，IT 人員容易遺漏，或因資訊不完整（如：忘記問電腦名稱、分機）而需反覆確認。
*   **溝通成本高昂**：描述電腦問題時，「畫面跳出一個視窗」往往難以精確傳達，導致維修判斷失準。
*   **缺乏數據管理**：報修紀錄散落在 Line、Email 或便利貼中，無法統計故障頻率，難以進行設備汰換評估。
*   **回應進度不明**：員工報修後不知道進度，產生焦慮與不信任感。

**資訊管家，正是為了徹底解決上述問題而生。**

---

## 3. 核心功能詳解

### 👨‍💻 使用者端：極致友善的報修體驗

1.  **標準化智慧表單**
    *   系統引導式填寫，確保所有必要資訊（姓名、工號、分機）一次到位。
    *   **遠端支援整合**：專屬欄位填寫 **Anydesk ID**，讓 IT 人員收到單據後能立即連線，實現「零接觸」快速維修。

2.  **強大的視覺化截圖工具 (V1.6 新增)**
    *   **多螢幕支援**：採用 `mss` 技術，完美支援雙螢幕或多螢幕工作環境，不會發生截圖錯位。
    *   **直覺選取**：滑鼠拖曳即可選取畫面上的錯誤訊息或異常區域，所見即所得。
    *   **即時預覽**：截圖或上傳圖片後，介面提供縮圖預覽，確保傳送正確的資訊。

3.  **安心的雙向回饋機制**
    *   提交成功後，系統會立即彈出確認視窗。
    *   同步發送**「報修受理通知信」**至員工信箱，讓員工知道需求已被重視。

### 🛠️ 管理端：高效透明的維運中台

1.  **自動化派單系統**
    *   報修單送出後，系統自動將需求發送至所有設定的 IT 管理員信箱。
    *   信件內容包含完整文字描述及高解析度附件圖片，IT 人員在手機上也能快速判讀問題嚴重性。

2.  **中央集權式資料庫 (FTP Sync)**
    *   **自動編碼**：系統自動生成每日唯一的報修單號（格式：`[公司代碼]YYYYMMDD-XXX`），便於追蹤與歸檔。
    *   **雲端同步**：所有報修紀錄自動彙整成 Excel 報表並上傳至企業內部的 FTP 伺服器，確保資料不因單一電腦故障而遺失。
    *   **數據分析基礎**：結構化的 Excel 資料可直接用於製作月報表，分析各部門報修頻率與常見故障類型。

3.  **多公司整合管理 (V1.6 新增)**
    *   **總報表系統**：單一介面即可切換檢視不同公司（揚發、萬陽、立侑螺絲、金總源）的報修紀錄。
    *   **狀態追蹤**：工程師可直接在系統上更新案件狀態（處理中、已結案）並回寫至資料庫。
